### Regenerating the wordlist

If you change `assets/imperatives.txt` or `assets/imperatives_blacklist.txt`, run

```bash
env SNAPSHOTS=overwrite cargo test
```
to regenerate the `wordlist_codegen.rs` file while running tests.
